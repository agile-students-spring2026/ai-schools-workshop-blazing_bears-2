# School District Evaluator

A web app that helps parents and educators evaluate school districts across the United States using publicly available data from the SchoolDigger API.

## Features

- Select a US state to view top-ranked school districts
- Click a district to see details and a list of schools within it
- Rankings, star ratings, and key stats at a glance

## Tech Stack

- React 18 + Vite
- Vitest + React Testing Library (100% code coverage)
- SchoolDigger API v2.3

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm

### Installation

```bash
git clone https://github.com/agile-students-spring2026/ai-schools-workshop-blazing_bears-2.git
cd ai-schools-workshop-blazing_bears-2
npm install
```

### Run the App

```bash
npm run dev
```

Then open http://localhost:5173 in your browser.

### Run Tests

```bash
npm test
```

### Run Tests with Coverage

```bash
npm run coverage
```

### Build for Production

```bash
npm run build
```
